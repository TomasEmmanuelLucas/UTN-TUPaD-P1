Proyecto de Funciones Básicas en Python

Qué se hizo
Se crearon varias funciones en Python para hacer operaciones sencillas y mostrar resultados al usuario. Las funciones son:

Tabla de multiplicar de un número.

Operaciones básicas (suma, resta, multiplicación, división) entre dos números.

Cálculo del índice de masa corporal (IMC) con peso y altura.

Conversión de grados Celsius a Fahrenheit.

Cálculo del promedio de tres números.

Cómo funciona
Cada función recibe los datos que ingresa el usuario, realiza el cálculo correspondiente y muestra el resultado en pantalla.

Cómo usar
Ejecutar el programa.

Ingresar los números o datos que se piden.

Ver el resultado impreso en pantalla.
