royecto de Funciones Recursivas
Qué hicimos
Hicimos programas que se llaman solos para resolver problemas.

Funciones que hicimos
es_palindromo:
Mira si una palabra se lee igual al derecho y al revés.

suma_digitos:
Suma los números que forman un número grande.

contar_bloques:
Cuenta cuántos bloques se necesitan para hacer una pirámide.

contar_digito:
Cuenta cuántas veces aparece un número dentro de otro número.

Cómo funciona
Cada función se llama a sí misma con un número más pequeño hasta que ya no puede seguir.

Ejemplos
es_palindromo("ana") → True

suma_digitos(123) → 6

contar_bloques(3) → 6

contar_digito(1223, 2) → 2

Lo que aprendí
Cómo hacer que una función se llame a sí misma.

Que siempre hay que tener una condición para parar.

Cómo usar matemáticas para trabajar con números sin convertirlos en texto.
