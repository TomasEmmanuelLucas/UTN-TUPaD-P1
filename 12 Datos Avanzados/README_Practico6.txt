Estructuras de Datos Complejas

Objetivo  
Practicar el uso de estructuras de datos complejas en Python para organizar y manipular información de manera eficiente.

Descripción  
Este práctico incluye ejercicios para trabajar con listas, tuplas, sets y diccionarios, aplicando operaciones comunes como agregar, modificar, consultar y combinar datos.

Uso  
Ejecutar los scripts en Python y seguir las instrucciones para interactuar con las estructuras de datos.
